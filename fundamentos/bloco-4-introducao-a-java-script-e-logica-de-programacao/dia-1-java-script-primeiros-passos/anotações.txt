DECLARAÇÃO A UMA VARIÁVEL:
- utilizamos "let"

ex.:
Atribuindo valor a uma variavel:    let age = 20 
Re-artibuindo valor a uma variavel:   age = 30

--------------------------------------------------

VARIÁVEL QUE NÃO ACEITA VARIAÇÃO:
- ultilizamos "const"

ex.; 
atribuindo valor a uma const:   const age = 20
(não da pra reatribuir um valor)

-------------------------------------------------

OPERADORES LÓGICOS

&& - and - Ele só vai retornar true se as duas operações que estão em volta dele forem consideradas verdadeiras
|| - or - Com o operador OR, precisamos apenas que uma das condições sejam verdadeiras: isso ou aquilo.
